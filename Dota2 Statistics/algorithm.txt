Hero Detail Page
    Features 
        Show hero info, abilities, facets, talents, matches, items, duration, match ups
    
    Components 
        HeroInfo + Innate + Facet + Abilities + Talents + Items + Matches + Duration + MatchUps

    HeroInfo 
        Show hero stats (str, agi, int, hp, mp , damage and so on)
        Show hero lore 

        Coding Logic
            Get hero id by using useParam()
            Get hero related data from heroes.json file 
            Get hero lore from lore.json file 

            Things that need to be calculated 
            hp -> base hp + (base str * 22);
            hp - regen -> base hp regen + (base str * 0.1)

            mp -> base mp + (base int * 12)
            mp - regen -> base mp regen + (base int * 0.05)

            min damage - > min damage + (primary attr)[primary attr mean if agi is primary attribute i have to add with base agi and so on]
            max damage -> max damage + (primary attr)

            for universal hero 
            min damage -> min damage + (base agi + base int + base str) * 0.45
            max damage -> max damage + (base agi + base int + base str) * 0.45 

            DON'T FORGET TO ROUNED TO TWO NEAREST DECIMAL PLACE hp regen and mp regen 
            and nearest integer for damage
